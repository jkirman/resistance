  • Game Rules
  1. Navigate to the Resistance game website (http://doorindustries-resistance.herokuapp.com)
  2. One player must first click the Create New Game button to create a new game room. That player will be directed to the waiting room with the game room’s link displayed.
  3. Send this link to all players in the game.
  4. Remaining players must then navigate to the room’s link.
  5. All players can then change their name to something of their choice.
  6. When a player is ready to begin the game, they must click the Ready button.
  7. When all players have clicked the Ready button, the game will automatically begin.
